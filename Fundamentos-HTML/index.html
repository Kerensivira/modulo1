-Fundamentos de una Página Web.-

Los elementos básicos para crear una página web son:

  1. HTML
  2. CSS
  3. JavaScript

los tres se encargan de distintas funciones dentro del funcionamiento de una aplicación o página web, como es el caso de recolección y almacenamiento de datos -Javascript-, estructura principal -HTML:5-, y estética de la página -CSS-. Hoy en día las páginas cuentan con gran contenido multimedia. Dicho en pocas palabras, una página web no es más que un documento digital multimediático, los formatos más comunes son:

  • Audio:
    • WAV
    • MIDI
    • AU -El más usado en las primeras páginas web-
    • MP3
    • Videos
      • AVI
      • MPEG
      • MOV
      • RealOne Player.
  • Imágenes
    • JPG
    • PNG
    • GIF
